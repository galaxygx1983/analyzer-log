# 概述

融合平台（ISCS）与运营零部件一体化系统（MPS）在控制中心有接口。本接口文件在双方招标文件有关融合平台（ISCS）与运营零部件一体化系统（MPS）的接口要求基础上进行细化，以指导双方系统深化设计、安装及调试工作，明确双方责任范围。

编制依据：《重庆市轨道交通24号线一期站后工程综合监控系统设备采购及集成服务项目合同》、重庆市轨道交通24号线一期站后工程融合平台系统用户需求书。

本接口规格书中涉及到的电线、电缆、光纤、熔接盒、光纤终端盒、尾纤等相关辅材及施工工作由施工单位提供与实施。

### 接口规格说明

#### 出库信息推送接口(ISCS-MPS-F01)

> 接口功能具体如下：

4.1-1 接口功能表

+------+------------------------------------------------------------------------------------------------------------------------------------+
| 序号 | 功能需求描述                                                                                                                       |
+:====:+:================================:+:===============================================================================================:+
| 1    | 接口编号                         | ISCS-MPS-F01                                                                                    |
|      +----------------------------------+-------------------------------------------------------------------------------------------------+
|      | 功能需求                         | 24号线ISCS融合平台相关物资，在运营零部件一体化系统由二级库出库至现场时，由MPS推送出库信息至ISCS |
|      +----------------------------------+-------------------------------------------------------------------------------------------------+
|      | ISCS实现功能                     | 接收出库信息（含退库信息），并更新现场库存                                                      |
|      +----------------------------------+-------------------------------------------------------------------------------------------------+
|      | MPS实现功能                      | 推送出库信息（含退库信息）                                                                      |
+------+----------------------------------+-------------------------------------------------------------------------------------------------+

- 输入：

  --------------------------------------------------------------------------------
  **序号**   **字段名称**            **字段说明**                  **类型**
  ---------- ----------------------- ----------------------------- ---------------
  1          IOP_PlanNO              出库明细项号                  String

  2          IOP_OrderNo             订单号                        String

  3          GD_NO                   物资编码                      String

  4          IOP_PlanCount           出库数量                      String

  5          IOP_CancelPerson        库管员                        String

  6          IOP_CancelDate          出库时间                      String

  7          IOP_ConditionCode       资金来源                      String

  8          GD_TT_NO                物资分类编码                  String

  9          GD_TT_Name              物资分类名称                  String

  10         GD_Name                 物资名称                      String

  11         GD_SpecificationsType   规格型号                      String

  12         GD_Unit                 计量单位                      String

  13         IOP_BatchNO             实物码                        String

  14         TrayNo                  托盘号                        String

  15         IOP_State               出库类型                      String

  16         IOP_SD_Name             供应商                        String

  17         IOP_ExpirationDate      失效期                        String

  18         MainDocumentNo          出库单号                      String

  19         WarehouseNo             仓库号                        String
  --------------------------------------------------------------------------------

- 输出：

  -----------------------------------------------------------------------
  **序号**   **字段名称**   **字段说明**                  **类型**
  ---------- -------------- ----------------------------- ---------------
  1          状态           1成功/-1失败                  int

  2          返回信息       字符串（接口状态信息）        String
  -----------------------------------------------------------------------

#### 退库申请推送接口（ISCS-MPS-F02）

> 接口功能具体如下：

4.1-1 接口功能表

+------+--------------------------------------------------------------------------------------------------------------------------------------------------------+

+:====:+:================================:+:===================================================================================================================:+

|      +----------------------------------+---------------------------------------------------------------------------------------------------------------------+

|      | ISCS实现功能                     | 推送退库申请信息                                                                                                    |
|      +----------------------------------+---------------------------------------------------------------------------------------------------------------------+
|      | MPS实现功能                      | 接收退库申请信息，并返回退库申请单号。                                                                              |
+------+----------------------------------+---------------------------------------------------------------------------------------------------------------------+

- 输入：

  --------------------------------------------------------------------------------
  **序号**   **字段名称**            **字段说明**                  **类型**
  ---------- ----------------------- ----------------------------- ---------------
  1          IOP_PlanNO              出库明细项号                  String

  2          IOP_OrderNo             订单号                        String

  3          GD_NO                   物资编码                      String

  4          IOP_PlanCount           出库数量                      String

  5          IOP_CancelPerson        库管员                        String

  6          IOP_CancelDate          出库时间                      String

  7          IOP_ConditionCode       资金来源                      String

  8          GD_TT_NO                物资分类编码                  String

  9          GD_TT_Name              物资分类名称                  String

  10         GD_Name                 物资名称                      String

  11         GD_SpecificationsType   规格型号                      String

  12         GD_Unit                 计量单位                      String

  13         IOP_BatchNO             实物码                        String

  14         TrayNo                  托盘号                        String

  15         IOP_State               出库类型(20-退库)             String

  16         IOP_SD_Name             供应商                        String

  17         IOP_ExpirationDate      失效期                        String

  18         MainDocumentNo          出库单号                      String

  19         WarehouseNo             仓库号                        String
  --------------------------------------------------------------------------------

- 输出：

  --------------------------------------------------------------------------------
  **序号**   **字段名称**   **字段说明**                           **类型**
  ---------- -------------- -------------------------------------- ---------------
  1          状态           1成功/-1失败                           int

  2          返回信息       字符串（接口状态信息含出库明细项号）   String
  --------------------------------------------------------------------------------
