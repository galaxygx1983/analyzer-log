# 概述

融合平台（ISCS）与轨道设备检修管理系统（RMS）在控制中心有接口。本接口文件在双方招标文件有关融合平台（ISCS）与轨道设备检修管理系统（RMS）的接口要求基础上进行细化，以指导双方系统深化设计、安装及调试工作，明确双方责任范围。

编制依据：《重庆市轨道交通24号线一期站后工程综合监控系统设备采购及集成服务项目合同》、重庆市轨道交通24号线一期站后工程融合平台系统用户需求书。

本接口规格书中涉及到的电线、电缆、光纤、熔接盒、光纤终端盒、尾纤等相关辅材及施工工作由施工单位提供与实施。
